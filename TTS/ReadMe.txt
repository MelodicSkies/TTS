--------------------------------------------------------------- Melodic Skies TTS Extension ---------------------------------------------------------------

Intended to be used in conjunction with the Melodic Skies Twitch Bot
Requires an Internet connection to function
No options for other voices

**Still a WIP**
-----------------------------------------------------------------------------------------------------------------------------------------------------------

Release V 1.0:
--------------
> basic text to speech bot that uses Google Translate to convert text into .mp3 files
> includes option to filter out messages with banned words
> strips emotes from messages to avoid spam

--------------------------------------------------------------------- Future Updates ----------------------------------------------------------------------

> direct integration with Melodic Skies Twitch Bot
> possible switch to another tts platform as Google Translate does not have additional voice options and requires the creation of .mp3 files for each request (could be problematic in high traffic chats)
> addition of an advanced spam filter to check for repetitive words/characters
> use Twitch API to check if a string is a registered emote (not sure if this is supported)
